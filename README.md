- Javascript

## Getting started

### Instalation

1. Clone the repository from github

```sh
git clone https://github.com/k-Dobosz/RecScan-API
```

2. Install dependencies

```sh
npm install
```

3. Enter your mongodb server url in `config/dev.env`

```
MONGODB_URL=YOUR_URL_HERE
```

3. Start the dev server

```sh
npm run dev
```
